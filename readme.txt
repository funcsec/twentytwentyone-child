== Description ==

Twenty Twenty-One Child is a child theme for the Wordpress theme Twenty Twenty-One.

== Installation ==

1. Add twentytwentyone-child to wp-config/themes/
2. Rename any fields you want in style.css 
3. Select Twenty Twenty-One Child from Appearance>Themes in the Wordpress backend
4. Build your masterpiece
